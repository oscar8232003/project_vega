Informacion y Categoria de ofertas

Ofertas Generales Premium

Este tipo de oferta es unico, solamente se puede tener 1 oferta general activada, esta oferta es la primera la cual se le muestra al cliente, puedes crear las que quieras pero solamente puedes tener 1 activada y es solamente para usuarios premium, no lo olvides!.

Ofertas de Temporadas Premium

Este tipo de oferta es la de temporada, no se especificara a cual temporada pertenecera para que esta seccion sea mas facil de manejar, usted como vendedor puede crear las ofertas de temporadas que quiera sin limite para su local, este beneficio es solo para clientes premium!.

-Ofertas por rango de cliente

Este tipo de ofertas va dirigido a ofertas por rango de cliente, puedes crear ofertas solamente para cliente que sean de distinto rango, ya sea plata, oro y diamante, cada rango se representada por la fidelidad a su local, por ende entre mas compre un cliente, mas beneficios le puede dar para incentivarlo a que siga con usted.

-Ofertas convencionales

Este tipo de ofertas, son ofertas comunes y corrientes, se pueden crear ilimitadamente y son propias de todo cliente.
